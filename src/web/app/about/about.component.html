Pretend You're Xyzzy Xyzzy 2 is a remake of Pretend You're Xyzzy and is in no way endorsed or sponsered by it's creator. This project is an attempt to update the UI of the original web app by using Angular, Material Design, and mobile first design principles. The original web app can be accessed here. License Xyzzy 2 is a Cards Against Humanity clone, which is available at cardsagainsthumanity.com, where you can buy it or download and print it out yourself.
